# Template NodeJs Web

**Install**
1. git clone 
```$xslt
git clone https://github.com/meellon/template.node.express4.git
```

2. npm install

```$xslt
cd template.node.express4
npm install
npm audit fix ( for version fix )
```

**Run**

```$xslt
nodemon
  or
npm start
```
